At AdobixTech, we prioritize your privacy. This Privacy Policy transparently explains how Limitly App-Block ("the App") handles your information with strict adherence to data protection standards.

1. Information Collection and Usage

The App operates with a strict no-data-collection principle:

  • Local storage only: All data remains exclusively on your device
  • Stored information:
    • Package names of blocked apps (no app content)
    • Your custom blocking schedules and settings
  • Zero data transmission: No internet connection required or used

2. Accessibility Service Usage (Full Disclosure)

The App requires Accessibility Service solely for these technical functions:

  • Detection: Identify when apps launch (package names only)
  • Blocking: Intercept attempts to open restricted apps
  • Functionality: Display time restriction enforcement prompts

Explicit Limitations:

  • No content access: Cannot and does not read messages, emails, or any in-app content
  • No tracking: Never records keystrokes, screenshots, or browsing activity
  • No data collection: Does not gather usage statistics or personal information
  • No network access: All processing occurs 100% on-device

3. SYSTEM_ALERT_WINDOW Permission

This permission enables only these visual functions:

  • Display blocking notifications when restricted apps are launched
  • Show time restriction reminders during blocked periods

Security Guarantee: The overlay cannot interact with other apps or extract any information.

4. Data Storage and Retention

Your complete data lifecycle protection:

  • Storage method: Android's encrypted local storage
  • Data types stored:
    • Blocklist (app package names only)
    • Custom restriction schedules
  • Automatic deletion: All data is permanently erased upon uninstallation
  • No backups: Never synced to cloud services (Google Drive, iCloud, etc.)
  • No cross-device transfer: Data never leaves your device

5. Security Measures

We implement multiple protection layers:

  • Android's built-in sandbox security for all stored data
  • No internet permission in manifest (technical enforcement)
  • Accessibility Service restricted to package name detection only
  • Regular security audits of all permission usage

6. Children's Privacy

The App complies with COPPA regulations:

  • Not designed for children under 13
  • No age-based data collection differences
  • Parental control features contain no data gathering

7. Policy Updates

We will:

  • Notify users of material changes via in-app alerts
  • Maintain version history with change dates
  • Never reduce your privacy protections without consent
